## Instructions

1. Go to your Render service dashboard: https://dashboard.render.com
2. Select your `sos-backend` service
3. Go to **Environment** tab
4. Click **Add Environment Variable** for each variable above
5. Copy the Key=Value pairs from the "Quick Copy-Paste Format" section above
6. For `CORS_ORIGIN`, use your actual backend service URL (e.g., `https://sos-backend-yffi.onrender.com`)

## ⚠️ IMPORTANT: Values You MUST Update

Before deploying, you **MUST** update these values:

1. **NANGO_SECRET_KEY** - Get from [nango.dev](https://nango.dev) (REQUIRED for OAuth connectors)
2. **OPENAI_API_KEY** - Get from [platform.openai.com](https://platform.openai.com) (if using OpenAI)
3. **ANTHROPIC_API_KEY** - Get from [console.anthropic.com](https://console.anthropic.com) (if using Anthropic)
4. **RESEND_API_KEY** - Get from [resend.com](https://resend.com) (replace `re_xxxxx`)

## Notes

- `REDIS_URL` will be automatically set if you're using Render's Redis service
- `JWT_SECRET` can be auto-generated by Render (set `generateValue: true` in render.yaml)
- `VITE_API_URL` should be left empty (frontend uses `window.location.origin` when empty)
- `NODE_ENV` and `PORT` are automatically set by Render
- Optional variables can be omitted if you're not using those services
